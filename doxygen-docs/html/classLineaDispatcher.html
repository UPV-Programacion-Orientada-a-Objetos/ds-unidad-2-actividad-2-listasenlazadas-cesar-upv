void LineaDispatcher::onRawLine ( const char *  linea)

Procesa una línea cruda del puerto serie.

Se ignoran todas las tramas hasta recibir la palabra clave "INICIO". Posteriormente, cada línea válida genera los logs correspondientes, crea la trama adecuada y la procesa.

Parámetros
lineaTexto recibido (sin incluir el salto de línea final).
